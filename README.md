# robosimu

Interactive robotics simulations that run in the browser. No install, no dependencies.

**Live:** https://robosimu.github.io

## Levels

| File | Concept |
|------|---------|
| `arm-kinematics.html` | 3-DOF planar arm — forward kinematics, joint angles, coordinate frames |
| `pick-and-place.html` | Inverse kinematics — click a target, arm solves and moves |
| `mobile-robot.html` | Differential-drive robot — PID control, odometry |
| `slam.html` | SLAM — robot builds a map while navigating unknown space |
| `mobile-manipulation.html` | Mobile base + arm working together |

Built with Canvas 2D. No libraries.
